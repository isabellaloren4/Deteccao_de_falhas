# Deteccao_de_falhas
Desalinhamentos em eixos de sistemas rotativos são uma das principais causas de falhas mecânicas em máquinas industriais. Detectá-los precocemente é essencial para evitar paradas não planejadas, reduzir custos de manutenção e garantir a segurança operacional.  Nesta competição, o desafio é desenvolver um modelo de aprendizado de máquina capaz de classificar sinais temporais de vibração como:  1 – Operação Normal  0 – Desalinhamento Horizontal. Os dados foram coletados de um banco de ensaios instrumentado com sensores de vibração e simulação de falhas, conforme descrito no banco de falhas da UFRJ.
